# muhies-website

## Prerequisites

- ruby 3.13
- Jekyll

## Running the websever

type `bundle exec jekyll serve --livereload`



you will then be able to access the site on you local machine by typing `localhost` or `http://127.0.0.1:4000/muhies-website/`
